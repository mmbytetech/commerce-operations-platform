## 🚀 Overview

Enterprise Operations Platform is a modern full-stack business solution designed to help organizations manage customers, products, sales, purchasing, reporting, and day-to-day operations from a single dashboard.

Built with a scalable architecture using React, Next.js, NestJS, PostgreSQL, and Docker, the platform is suitable for startups, SMEs, and enterprise environments.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/mmbytetech/commerce-operations-platform.git
cd commerce-operations-platform
```

### Install dependencies

```bash
npm install
```

### Configure environment

```bash
cp .env.example .env
```


npm run dev
```

Open:

```
http://localhost:3000
```
